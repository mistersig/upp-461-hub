Start here

Welcome to UPP 461. This week introduces why geography matters for planning, what GIS is (and isn’t), and how to set yourself up for success in the course.

Estimated time: 20–35 minutes.

What you’ll learn

  • Define geography and explain why location matters
  • Explain Tobler’s Law in plain language
  • Describe GIS as both communication and analysis
  • Understand the basics of georeferencing
  • Apply course data management practices (naming rules, folder structure)

What you’ll do

  • Use a slider to visualize Tobler’s Law (distance decay)
  • Sort examples into “communicative” vs “analytical” GIS uses
  • Check a filename/path against ArcGIS-friendly naming rules
  • Take a quick quiz (ungraded, for feedback)

Quick setup reminder

ArcGIS Pro is Windows-only. If you use macOS, plan to use the Virtual Lab or a Windows setup. Start building good file habits now—GIS projects can break if you lose track of your data.
