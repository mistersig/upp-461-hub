Data management

Data management (you will thank yourself later)

ArcGIS Pro + macOS

  • ArcGIS Pro runs on Windows.
  • macOS options: (1) install Windows via virtualization (e.g., Parallels), or (2) use Remote Desktop to the Virtual Lab.
  • If you use campus lab computers and your own machine, bring a USB-A/USB-C drive for moving data.
Reminder: We do not provide technical support for macOS setup — use the Virtual Lab if you need a reliable path.

The rules for naming GIS files and folders (ArcGIS-friendly)

  • No spaces (use underscores).
  • No special characters like commas, periods, !, @, # (underscores are OK).
  • Begin names with letters, not numbers.
  • Rules apply to the entire path (parent folders too).
Good example: C:\GIS\UPP461_Fall2026\Assignment1\MyData\oakparktracts.gdb

Workspace & organization (recommended)

  1. Create a root folder for the course: C:\GIS\UPP461_YourName
  2. Inside, make one folder per assignment/lab/week.
  3. Keep raw downloads separate from cleaned/processed data.
  4. When in doubt: duplicate the data before major edits.
