Geoprocessing tools

Geoprocessing is spatial data processing: you run GIS operations that transform input layers into new outputs for analysis or preparation.

Three categories you should recognize

Extraction

Reduce your dataset to the relevant area/features.

Example: Clip

Overlay

Combine layers based on overlap.

Examples: Intersect, Union

Proximity

Relationships based on distance.

Example: Buffer

Clip (extraction)

Clip is the “cookie cutter” tool: it extracts features within the extent of a clip layer and reduces the dataset.

Clip often improves cartography (“clean edges”), but be careful when analysis needs full features/attributes.

Clip vs Select by Location

Check yourself: when is Select by Location better?

If you need the full geometry and attributes of selected features (e.g., geocoding workflows, network analysis inputs), selecting may be preferable.

Dissolve (generalization / aggregation)

  • Combines adjacent polygons based on a shared field (fewer features).
  • Often used to create regions and summarize attributes (e.g., sum population).

Buffers (proximity)

Buffers create zones around points/lines/polygons to represent influence, access, or exposure within a distance.

Overlay: Intersect (AND) vs Union (OR)

Check yourself: which tool would you use?

Goal: Find buildings exposed to flood zones.

Likely tool: Intersect (AND) to keep only overlapping building areas.
