Quick quiz

Quick quiz: Check your understanding of spatial analysis concepts and core GIS operations.

Nothing is graded here; use it to identify what to review before class/lab.

1) Which best describes a spatial question?
Spatial questions depend on location, distribution, association, interaction, or change—so spatial relationships affect the answer.
The key feature is that space/location changes the answer. Numbers or spreadsheets alone don’t make a question spatial.
2) Which geoprocessing category does Buffer belong to?
Buffers create zones based on distance, which is a proximity relationship.
Extraction reduces features (clip). Overlay combines layers by overlap (intersect/union).
3) Clip is often described as a “cookie cutter” tool because it…
Clip outputs only the input features within the clip boundary—reducing the dataset and creating clean edges.
Those other actions are different tools/methods (joins, networks, hotspots).
4) Intersect and Union differ mainly because Intersect uses AND logic and Union uses OR logic. Which statement matches that?
Intersect = overlap (AND). Union = all parts from both inputs (OR).
Reversing the logic will give very different outputs and often the wrong answer for policy questions.
5) Dissolve is most appropriate when you want to…
Dissolve generalizes/aggregates polygons using a common field and can summarize attributes (e.g., sum population).
Clip cuts, proximity tools measure distance/buffers, and attribute joins use matching IDs.
6) A core rule of spatial joins is: whatever feature type you start the spatial join with, you end with…
Spatial joins produce an output of the target layer type, with new fields joined/aggregated from the join layer.
Points/polygons aren’t guaranteed; it depends on which layer is the target.
7) You want to count how many clinics fall within each community area polygon. What should be the target layer in a spatial join?
If you want results per community area, make community areas the target so the output is polygons with a clinic count field.
If clinics are the target, you’d get point output (one row per clinic), not one row per community area.
8) Why can spatial joins fail near Lake Michigan when joining Chicago census tracts to community areas?
Geometries that extend into “no data” areas can break simple spatial rules. You often need to clip first or choose a defensible rule and validate.
The failure is geometric/logic-based, not because Chicago data lacks geometry or because buffers are impossible.
9) If your real question is “within a 15-minute transit ride,” what method is usually more appropriate than a simple Euclidean buffer?
Travel time depends on the network (roads/transit), not straight-line distance, so network analysis is the better fit.
Clip/dissolve/union don’t model travel time; they manipulate geometries but don’t represent movement constraints.
10) Which is an example of a “natural experiment” in policy evaluation?
Natural experiments arise from real-world variation not controlled by the researcher (across places or time).
Random assignment is a controlled experiment; map design is not a causal evaluation method.
