What is spatial analysis?

Spatial analysis is the set of methods we use to answer questions where location and spatial relationships matter.

In practice, you’re turning spatial data into evidence that can support a planning or policy decision.

Five types of geographic questions

GIS helps answer questions about:

  • Location: Where is it? Why is it here?
  • Distribution: Clustered or dispersed? Where are boundaries?
  • Association: What else occurs with it (or is absent)?
  • Interaction: How much movement/connection occurs between places?
  • Change: How does it change over time and space?

What makes a question “spatial”?

A spatial question is one that depends on the spatial dimensions of a phenomenon, and can be answered using spatial analysis techniques.

If you removed the map and the answer would be the same, it might not be spatial.

Check yourself: classify the question

Question: “Are air-quality complaints clustered near major arterials?”

Type: Distribution (clustered vs dispersed) + Association (near arterials).

From question → method

Common translation patterns:

  • “Near” → Buffer / Within a distance
  • “Inside” / “overlaps” → Intersect / Within
  • “Combine areas by category” → Dissolve
  • “Count points in polygons” → Spatial join (aggregate/count)
