Layout elements

Core map elements

Most layouts combine:

  • Title (descriptive: what/where/when)
  • Legend (only what needs interpretation)
  • Scale (bar/ratio/verbal) when distances matter
  • Credits & data source (who made the data/map)
  • North arrow when orientation isn’t obvious

Try it: choose map elements

Some maps intentionally omit elements (e.g., transit diagrams omit scale). Choose what makes sense for each scenario.

Include these elements

Why this matters

  • Missing title = reader confusion.
  • Missing credits = untrustworthy map.
  • Legend overload = wasted attention.

Scale: three common forms

  • Representative fraction: 1:10,000
  • Scale bar: graphic bar
  • Verbal: 1 inch = 500 miles
