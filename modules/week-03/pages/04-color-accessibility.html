Color + accessibility

Color basics

  • Hue = the basic color (red, blue, green…).
  • Value = lightness/darkness (often more important for readability).

Rule of thumb: categorical data → different hues; quantitative data → same hue with different values/saturation.

Accessibility reminders

  • Don’t rely on color alone — use labels, patterns, or outlines.
  • Avoid red–green ramps for diverging data (common color-blind issue).
  • Consider cultural meaning and mapping conventions (e.g., red = risk).
