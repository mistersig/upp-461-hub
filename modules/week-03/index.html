What you’ll be able to do

  • Explain graphicacy and why it matters in GIS communication.
  • Apply figure–ground, contrast, and hierarchy to make the map’s message obvious.
  • Choose color schemes that match the data type (categorical vs quantitative) and remain accessible.
  • Build a layout using core map elements (title, legend, scale, credits) and justify what you include or omit.

This module supports lecture and in-class activities. It’s not a substitute for attending class.

How to use this module

  1. Work through pages in order (left nav or Next buttons).
  2. Try the interactives — they’re designed to reveal common mistakes.
  3. Take the Quick Quiz at the end for a score + explanations.

Key idea

Visual design in maps is about directing attention and controlling focus.

Your map is an argument: what’s important should “pop,” and everything else should support it without competing.
