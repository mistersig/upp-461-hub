Quick quiz

Quick quiz: Use this to check your understanding before class.

Nothing is graded here; focus on the explanations.

1) Which statement best describes a raster dataset?
Rasters are gridded cells (pixels) storing values (categories or measurements).
Vector features and networks are different data models. Raster = grid of cells.
2) If you decrease raster cell size (higher resolution), what usually happens?
Smaller cells increase cell count, which increases storage and compute cost.
Higher resolution generally means more cells, not fewer.
3) “NoData” in a raster most accurately means…
NoData represents missing/undefined values; 0 is a real value if 0 is meaningful in that dataset.
NoData is not “zero.” Tools often treat NoData specially, but behavior varies.
4) Reclassification is best described as…
Reclass remaps values/ranges into new categories or scores.
Projection changes and smoothing are different operations.
5) When resampling a categorical land cover raster, which method is usually appropriate?
Nearest neighbor preserves category codes and avoids creating “mixed” classes.
Bilinear/cubic create new intermediate values, which is bad for categories.
6) A focal operation uses…
Focal (neighborhood) tools compute outputs from a moving window around each cell.
Local uses just the corresponding cell; global uses the whole raster; zonal summarizes within zones.
7) In IDW interpolation, increasing the power parameter (p) tends to…
Higher p increases distance decay, so nearby points dominate.
IDW still depends on measured points; it doesn’t make influence equal or favor far points.
