Summary

Summary

  • Raster represents space as a grid of cells; each cell stores a value.
  • Resolution (cell size) is a tradeoff: more detail vs. more storage and compute.
  • Map algebra is “grid math” — unary, binary, and conditional overlays.
  • Reclassification remaps values into new classes (an analytical choice).
  • Resampling changes cell size/alignment; choose methods based on data type.
  • Raster operations have a scope: local, focal, zonal, or global.
  • Interpolation (e.g., IDW) estimates values between measured points; parameters matter.
