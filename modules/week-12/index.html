Start here

What you’ll learn: how raster data stores information, how raster processing works (map algebra, reclassification, resampling), and how continuous surfaces can be created with interpolation (IDW).

  • Recognize the raster data model and its key properties (cell size, extent, NoData, attribute tables).
  • Explain map algebra and common raster operations (local/focal/zonal/global).
  • Choose appropriate reclassification and resampling approaches.
  • Describe interpolation and how IDW works (including the effect of the power parameter).

Estimated time: 20–30 minutes. Bring questions to class.

Key idea

Raster analysis is “math on grids.” Once you understand how cells represent the world, many tools become variations of the same few patterns.

  • Local: cell-by-cell
  • Focal: neighborhood window
  • Zonal: summarize within zones
  • Global: whole raster
