What you’ll learn: how attribute tables store data about features, how to write queries (SQL / “where clauses”), and how to connect datasets with joins/relates.

  • Read records (rows) and fields (columns) confidently.
  • Write simple and compound queries using comparison + boolean operators.
  • Use Select by Attribute and Select by Location
  • Explain and troubleshoot joins and relates.

Estimated time: 20–35 minutes.

Three core moves

  • Query: Which features meet criteria?
  • Spatial query: Which features are near/inside/overlapping?
  • Join: What extra attributes can I attach?

Tip

If a join “fails,” the key field almost always has a mismatch (type, leading zeros, spaces, or duplicates).
