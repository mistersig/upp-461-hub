Attribute tables

Attribute tables = your data engine

Attribute tables are the spreadsheet view of a layer. You’ll query, sort, summarize, and join from here.

  • Row = record = one feature
  • Column = field = one attribute type
  • Cell = the value for a feature and field

Field types matter

  • Text: names, IDs (use for codes with leading zeros)
  • Integer: counts
  • Float/Double: rates and measurements
  • Date: timestamps

Join keys

For joins/relates, key values must match in type and format (zeros, spaces).
