SQL + where clauses

SQL / “where clauses”

A where clause selects rows using comparisons and boolean logic.

  • Comparisons: =, >, <, >=, <=, !=
  • Boolean: AND, OR, NOT
  • Pattern: LIKE (varies by platform)

Remember: text values usually need quotes (e.g., '031').
