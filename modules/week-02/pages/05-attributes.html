Attributes & measurement scales

GIS features are usually stored as geometry (coordinates) + attributes (fields/values).

Measurement scales (why they matter)

A variable’s scale affects what operations make sense.

Scale What it means Safe operations Example
Nominal Categories with no order Counts, mode Land cover class
Ordinal Ordered categories (spacing unknown) Ranks, median, percentiles “Low / Medium / High” index
Interval / Ratio Numeric with meaningful differences (and often a true zero) Arithmetic (mean, sum), modeling Population density, income, elevation
Common mistake: averaging category codes (e.g., land cover 1–8) as if they were measurements.
