Summary

Key takeaways

  • Representation is a choice. Vector and raster both simplify the world—know what assumptions you inherit.
  • Topology rules protect analysis. Gaps/overlaps/misalignment can create bogus results; not every mismatch is equally important.
  • Measurement scales matter. Don’t treat categories like measurements.
  • Data structures and file formats matter. Shapefiles are fragile clusters; geodatabases are built for management.
  • Metadata is your safety net. If you can’t explain provenance/accuracy, be cautious in interpretation.

If you remember one thing…

Ask “How was this data made?” before you trust the output of an analysis tool.
