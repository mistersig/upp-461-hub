Quick quiz

Quick quiz: Check your understanding before class.

Nothing is graded here; focus on the explanations.

1) Which data model is usually best for representing elevation as a continuous surface?
Elevation varies continuously; rasters support surface and neighborhood operations.
Vector points can sample elevation, but they do not represent a continuous surface by default.
2) “Roads as lines vs roads as areas” mainly depends on…
Representation changes with map scale and task (centerlines vs right-of-way).
Basemaps and file extensions don’t determine the correct conceptual model.
3) A common planar topology rule for a polygon partition is…
Overlaps can double-count area and break partition assumptions.
Shape and area uniformity are not required; rasters are a different data model.
4) Layers don’t align perfectly. Before calling it an “error,” you should consider…
Misalignment can be “faux” due to different production methods and scales.
Display choices don’t explain production differences between datasets.
5) Land cover class (forest/water/impervious…) is best described as…
These are categories; numeric codes are labels, not measurements.
There is no inherent numeric order or meaningful distance between categories.
6) In a shapefile “cluster,” which file commonly stores attributes?
The .dbf holds the attribute table; the .shp holds geometry.
.prj is projection info; .shx is an index; geometry lives in .shp.
7) The best reason to care about metadata is that it helps you…
Metadata helps you evaluate whether a dataset is appropriate for a decision or analysis.
Metadata doesn’t change system performance or replace technical understanding.
