SRS MODULE (Web) — WCAG-minded, light/dark themes

Open locally:
1) Unzip the folder.
2) Open index.html in a browser.

Web-map activity (part of the module):
- After “Projections,” open “Web maps” in the navigation.
- This activity uses Leaflet + OpenStreetMap tiles and requires internet access.

Theme:
- Light is default.
- Students can toggle Dark with the Theme button (top right).
- Both themes use high-contrast, color-blind-friendly accents.

DISCLAIMERS
- For educational purposes only.
- Not for navigation, surveying, engineering design, or legal boundary determination.
