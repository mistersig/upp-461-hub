Key takeaways

  • CRS vs projection vs datum: a Coordinate Reference System (CRS) includes both the coordinate system and how it relates to the Earth.
  • GCS vs PCS: Geographic Coordinate Systems use latitude/longitude; Projected Coordinate Systems use linear units (meters/feet) on a flat map.
  • Web Mercator is a choice: it’s common for web maps, but not always appropriate for measurement or analysis.

Common pitfalls to avoid

  • Mixing layers with different CRS without reprojecting correctly.
  • Measuring distance/area in a CRS that distorts the results.
  • Assuming a basemap’s CRS matches your data.
