6) State Plane (example: Illinois)

State Plane Coordinate System (SPCS) is designed for high-accuracy local mapping within U.S. states. It uses zones to keep distortion small.

  • Units often in feet or meters (depends on definition)
  • Multiple zones per state
  • Great for engineering / cadastral / local government mapping

Mini activity

Why use State Plane instead of Web Mercator for local analysis?

State Plane is designed to minimize distortion locally and uses linear units, making measurements and analysis more accurate for local areas.
