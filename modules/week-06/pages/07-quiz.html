Quick quiz

Try these questions. Your feedback will include:

  • Why a wrong choice is wrong
1) What is the best description of a datum?

A datum defines the Earth model (ellipsoid + alignment) that your coordinate system references.
A projection is the flattening method (not a datum), and file formats are unrelated to datums.
2) Which is true about a Geographic Coordinate System (GCS)?

A GCS uses angular units (degrees) like latitude and longitude on a curved surface.
Meters are typical of projected systems, and preserving area is a projection property, not a definition of GCS.
3) You need accurate area comparison for counties. What property do you prioritize?

Equal-area projections preserve area, which is essential for fair comparison of region sizes.
Conformal projections preserve local shape/angles, not area. “Looks best” can hide major quantitative distortion.
4) If a dataset is missing CRS information, what should you do first?

Defining the CRS tells the software what the existing coordinates mean. Projecting without a correct definition can create wrong results.
Projecting without knowing the source CRS is guessing; deleting the data doesn’t solve the underlying metadata problem.
