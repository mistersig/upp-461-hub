Quick quiz

Quick quiz: Check your understanding of choropleth mapping choices.

Nothing is graded here; focus on the explanations.

1) A sequential color ramp is best for…
Sequential ramps encode magnitude in a single hue (light→dark).
Categorical uses different hues; diverging is for midpoint comparisons.
2) A diverging ramp is most appropriate when…
Diverging ramps emphasize direction away from a central value.
More colors isn’t better; categories need distinct hues; totals need normalization.
3) Quantile classification guarantees…
Quantile forces equal counts per class.
Equal interval is equal-width; outliers can still exist; breaks depend on the data.
4) Equal interval classification can struggle when…
Skew can push most features into 1–2 bins and leave empty bins.
Normal distributions are usually fine; the issue is skew and empty bins.
5) Why avoid mapping raw counts on different-sized areas?
Normalize to rates/densities so the map compares like with like.
The core issue is comparability—counts track size more than intensity.
6) A good default number of classes for choropleths is…
Humans can reliably distinguish only a limited number of shades.
Too many classes reduces readability.
7) Which is an example of normalization?
Rates compare areas fairly by using a denominator.
Totals are not normalized by a denominator.
